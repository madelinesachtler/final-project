Overview

The final chart located on this dashboard compares the average number of sabotages fixed to whether the imposter was ejected or not. The quicker an imposter is ejected, the greater number of crewmates there are to fix sabotages. So statistically speaking, the average number of sabotages fixed should increase with each imposter ejected. Similarly, our chart displays just that. The average number of sabotages fixed increases to 0.5636 from 0.4291 once the imposter(s) are ejected.
